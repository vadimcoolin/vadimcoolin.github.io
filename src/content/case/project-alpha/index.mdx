## Overview

Project Alpha is a team workflow management tool built to streamline task assignment, progress tracking, and team communication in one unified interface.

## Key Features

- **Real-time Collaboration** — Built with WebSocket for instant updates across team members
- **Task Management** — Kanban-style boards with drag-and-drop reordering
- **Team Chat** — Integrated messaging with thread support

## Technical Highlights

The frontend is built with React and TypeScript, using RTK Query for efficient data fetching and caching. The backend runs on Node.js with Express, backed by PostgreSQL for persistence.

WebSocket connections are managed through a pub/sub pattern, ensuring that only relevant updates reach each connected client.

## Outcome

The application reduced team coordination overhead by approximately 40% and has been adopted by three teams within the organization.
